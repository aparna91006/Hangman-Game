Hangman Game

A console-based Hangman game developed in Python as an individual project.

Features

- Loads words from an external text file using file handling.
- Randomly selects a word for each game.
- Adapts the number of attempts based on the length of the selected word.
- Validates user input.
- Tracks guessed letters during gameplay.
- Reveals all occurrences of correctly guessed letters.

Technologies Used

- Python
- File Handling
- Random Module

Project Structure

- "hangman.py" – Main game logic
- "choose_word.py" – Random word selection
- "extracting_hangman_reqd_words.py" – Prepares the word list for the game
- "words_reqd_hangman.txt" – Word dataset

How to Run

1. Ensure all project files are in the same directory.
2. Run "hangman.py".
3. Enter one letter at a time to guess the hidden word.
